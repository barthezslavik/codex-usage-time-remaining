## Summary

## Description

Small Chrome extension for the ChatGPT Codex analytics page.

It keeps the original green quota bar and adds a second blue bar below it. The blue bar shows how much time is left in the current 7-day weekly cycle, based on the reset time displayed by ChatGPT.

Useful if you want to compare:

- quota remaining;
- time remaining until reset;
- whether you are spending Codex quota faster or slower than the weekly pace.

Install:

1. Download the ZIP.
2. Unzip it.
3. Open `chrome://extensions`.
4. Enable `Developer mode`.
5. Click `Load unpacked`.
6. Select the unzipped folder.
7. Refresh `https://chatgpt.com/codex/cloud/settings/analytics#usage`.
