# Release Checklist

## GitHub

1. Create a public repo named `codex-usage-time-remaining`.
2. Push this project.
3. Add a release named `v1.0.0`.
4. Attach `codex-usage-time-remaining.zip` to the release.
5. Enable Issues.
6. Fill sidebar settings from `github-repo-settings.md`.

## Ko-fi

1. Set your Ko-fi profile text as a support page for the extension.
2. Add a post using `kofi-post.md`.
3. Optional: create a pay-what-you-want Shop item with `codex-usage-time-remaining.zip`.
4. Link the GitHub release page.

## Promote

Use `promotion-plan.md` for launch copy.
