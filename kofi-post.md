## Title

Free tiny Chrome extension for ChatGPT Codex usage

## Body

I made a tiny Chrome extension for the ChatGPT Codex usage page.

It adds a second progress bar below the weekly quota bar:

- green = quota remaining;
- blue = time remaining in the 7-day reset cycle.

This makes it easier to see whether you’re using Codex quota faster or slower than the weekly pace.

The extension is free and open source on GitHub:

https://github.com/barthezslavik/codex-usage-time-remaining
